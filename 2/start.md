# Start

### 1. Install Python

```
python --version.
```

### 2. Set Up a Virtual Environment

```
python -m venv venv
```

#### Activate the Virtual Environment

*   **Windows**:

    ```
    venv\Scripts\activate
    ```
*   **Mac/Linux**:

    ```
    source venv/bin/activate
    ```
