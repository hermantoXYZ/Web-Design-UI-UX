# \<blockquote>

The `<blockquote>` tag in HTML is used to indicate a block of quoted text from another source. It helps improve readability and accessibility while also giving proper attribution to the original content.

#### **Basic Syntax:**

```html
<blockquote>
    "The only limit to our realization of tomorrow is our doubts of today."
</blockquote>
```

#### **With Citation (`cite` attribute):**

```html
<blockquote cite="https://example.com">
    "Success is not final, failure is not fatal: It is the courage to continue that counts."
</blockquote>
```

#### **Styling Example (Using CSS):**

```html
<style>
    blockquote {
        font-style: italic;
        border-left: 4px solid #ccc;
        padding-left: 10px;
        margin: 10px 0;
    }
</style>

<blockquote>
    "Innovation distinguishes between a leader and a follower."
</blockquote>
```

#### **When to Use `<blockquote>`:**

* Quoting a paragraph or multiple lines from another source.
* Displaying testimonials or reviews.
* Highlighting important text in an article.
